# fcv-technical

Como primera instancia se creo un repoitorio en git con el comando `git init`

Luego con la ayuda de la pagina https://start.spring.io/ se creo la estructura base del proyecto. Se creo la carpeta backend, se movio el proyecto al repositorio y se realizo el push al repositorio remoto. Se creo una rama nueva con el comando `git checkout -b feat/initial-back` y se realizo el push de la rama al repositorio remoto.
